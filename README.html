Alfaerie Omega Chess Variant Image Package v1.2

This directory contains graphics from the excellent Alfaerie Chess Variant Graphics set, plus expansion sets. The goal is to include all known Alfaerie chess graphics into one, flat archive. Instead of having separate downloads to add new images, a new version of this package can be released when there are new images. Contributions are welcome! Note that I didn't make most of these images, I'm merely collecting and organizing them.

Sets included here:

  • Original Alfaerie Variant Chess Graphics Set (source) by David Howe
  • Alfaerie Expansion Set One (source)
  • Alfaerie Expansion Set Two (source)
  • Alfaerie Expansion Set Three (source)
  • Alfaerie Expansion Set Four (source)
  • Alfaerie Plus (source), with a few minor modifications necesssary to avoid name conflicts (details)
  • Alfaerie Misc (source), except for identical images, and a few images had to be renamed to avoid conflicts also (details)
  • Alfaerie Expansion Set Five (source)
  • Alfaerie Expansion Set Six (source)
  • Alfaerie Beta contains new piece images not found elsewhere, including my humble contributions.

For an auto-generated list of the 756 available blue and white pieces, see the master list. For all images, see the index.

In general, images are named beginning with b or w, for blue or white, followed by a descriptive name for the piece. In some cases, g, r, or k prefixes are seen for green, red, or black. However, the blue pieces are intended to be used for the black pieces in chess. Only transparent gif images are included here, not Windows bitmaps.

Change Log

v1.2 (2010/11/04; 699 pieces) adds 20 new and exclusive piece images ("Alfaerie Beta"), and is the first public release, now available on GitHub.

v1.1 (2009/06/13; 679 pieces) renamed the filenames to be all lowercase, and use an underscore instead of a dash. Only affected a few alfaeriemisc images, mostly from Shogi. This change allows the piece filenames (minus extension) to easily be used as identifiers in many programming languages.

v1.0 (2009/06/12; 679 pieces) contained the original alfaerie set, expansion sets 1 to 4, alfaerie plus, and alfaerie misc, in a flat directory structure, with HTML files referencing each file. If you only want the original images, this is the version for you.
