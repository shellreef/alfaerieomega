Alfaerie Omega Chess Variant Image Package v1.2

This directory contains graphics from the excellent Alfaerie Chess Variant Graphics set, plus expansion sets. The goal is to include all known Alfaerie chess graphics into one, flat archive. Instead of having separate downloads to add new images, a new version of this package can be released when there are new images. Contributions are welcome! Note that I didn't make most of these images, I'm merely collecting and organizing them.

Sets included here:

  • Original Alfaerie Variant Chess Graphics Set (source) by David Howe
  • Alfaerie Expansion Set One (source)
  • Alfaerie Expansion Set Two (source)
  • Alfaerie Expansion Set Three (source)
  • Alfaerie Expansion Set Four (source)
  • Alfaerie Plus (source), with a few minor modifications necesssary to avoid name conflicts (details)
  • Alfaerie Misc (source), except for identical images, and a few images had to be renamed to avoid conflicts also (details)
  • Alfaerie Expansion Set Five (source)
  • Alfaerie Expansion Set Six (source)
  • Alfaerie Beta contains new piece images not found elsewhere, including my humble contributions.

For an auto-generated list of the 800+ available blue and white pieces, see the master list. For all images, see the index.

In general, images are named beginning with b or w, for blue or white, followed by a descriptive name for the piece. In some cases, g, r, or k prefixes are seen for green, red, or black. However, the blue pieces are intended to be used for the black pieces in chess. Only transparent gif images are included here, not Windows bitmaps.

Change Log

v2.0 (2011/02/26; 800 pieces) is a major revision, breaking backwards-compatibility

  • Added new pieces by Christa Connelly: blowfish, dog, dolphin, duck, flamingo, and witch
  • Added other new pieces: bronotosaurus, rat, mouse, fieldmouse, sidewinder, daemon, turtle, bison, buffalo, and many other variations
  • Added a total of 44 new pieces
  • Added a piece image wishlist (contributions welcome!)
  • Added index.jsonp as an alternative to loading index.json using Ajax
  • Added several informational tags to various pieces in index.json, and its keys are now sorted
  • Added a compact view to browse.html
  • master.html now displays pieces much more compactly, two-dimensionally instead of linearly
  • process_images.py is now compatible with Python 2.6
  • Consistency: the snipers forwardknightbackwardsprince and forwardrookbackwardsprince have been renamed to fknightbprince and frookbprince
  • Consistency: pieces digit0 through digit9 have been added, replacing 0-9.gif (edigit0-9.gif are the closest to the originals)
  • Consistency: caged pieces now confirm to the color prefix convention (bc?bishop is now ?cbbishop, etc.)
  • Consistency: the cage piece is now a solid color, instead of solid black
  • Removed rotated image generation by default, it produced too many files and is best handled within the app instead (but you can re-enable it in process_images.py if needed)
  • Removed black (k*.gif) pieces, too ugly (again, generation can be re-enabled in process_images.py if you miss these)
  • Removed highpriest_ess, which was identical to highpriestess2

v1.5 (2010/11/10; 756 pieces) has a bunch of improvements, notably:

  • incorporated Alfaerie Expansion Sets 5 and 6
  • added a handful of pieces with missing colors (smallcircle, forfnibakking, furlrurlbakking, wwall, ewall)
  • a new JSON piece index file with metadata about each piece (issue #6)
  • generated rotated versions (45°, 90°, 135° clockwise/counterclockwise, and inverted) of every piece in an optional package (issue #3)
  • generated yellow, red, green, cyan, orange, pink, and grey versions of every piece (issue #1)
  • Python script process_images.py for generating list.txt/master.html (supersedes the Perl script), updating index.json, and generating colored/rotated variants
  • an experimental Ajax-based piece browser for searching pieces by name or tag

v1.2 (2010/11/04; 699 pieces) adds 20 new and exclusive piece images ("Alfaerie Beta"), and is the first public release, now available on GitHub.

v1.1 (2009/06/13; 679 pieces) renamed the filenames to be all lowercase, and use an underscore instead of a dash. Only affected a few alfaeriemisc images, mostly from Shogi. This change allows the piece filenames (minus extension) to easily be used as identifiers in many programming languages.

v1.0 (2009/06/12; 679 pieces) contained the original alfaerie set, expansion sets 1 to 4, alfaerie plus, and alfaerie misc, in a flat directory structure, with HTML files referencing each file. If you only want the original images, this is the version for you.
